# Pokemon Marketplace

Pokemon Marketplace adalah aplikasi marketplace sederhana berbasis Flutter yang menggunakan Firebase sebagai backend. Aplikasi ini memungkinkan pengguna untuk melakukan registrasi dan login akun, melihat daftar produk Pokémon, menambahkan produk ke keranjang, serta melakukan checkout.

## Fitur

- Sign Up (Registrasi akun)
- Login akun
- Melihat daftar produk
- Menampilkan product card
- Menambahkan produk ke cart lewat firebase
- Menghapus produk dari cart
- Checkout produk
